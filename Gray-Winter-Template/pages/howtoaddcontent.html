How To Add Content To This Template

Before reading this page, please view the page How to Customize This Template.

After the template has been downloaded and customized, it is time to add content to the pages. On the about.html page, users can see a demo for all features. It is possible to simply cut and paste features (such as the gallery feature) from this page onto custom content.

Although I have designed this template to be easy to use, I understand that editing HTML and other web development tasks can be difficult for some people. That is why I have included several demo pages which exist for specific purposes. A user can simply change the images, text and link paths on these demo pages to add content and customize if they have a specific need.

Each HTML document has slightly different architecture based on the purpose of the page, but there is only a small amount of variation from page to page. It is possible to write the HTML document by hand, or copy and paste from any of the existing pages, or from any of the demo pages.

Once the pages have been uploaded and customized, adding content is as simple as cut and paste.

This template is relatively simple to customize and edit for someone with a basic knowledge of HTML. For more information on HTML5, please visit W3Schools.
