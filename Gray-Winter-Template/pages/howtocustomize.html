How To Customize This Template

Before reading this page, please read How To Download This Template.

This template uses CSS and javascript, but most of the design aspects are controlled with CSS. Each part of the template is styled using a separate CSS file, all of which are called using one stylesheet and the @import at-rule. All of the necessary CSS files are called within one file called /style.css which is located within the assets/template/css/ subfolders.

It is not necessary for the user to touch the main stylesheets in order to customize this template. When customizing this template, a user should not open the /template/ folder or touch any of the files inside. Instead, the user should open the /custom/ folder within the /assets/ folder. Inside, there is a file called /custom.css. The user can then add custom CSS to this file in order to overwrite default template styles. From the /custom.css file, the user may add or remove their own background images and customize the color palette. Everything that the user writes in the /custom.css file will overwrite the template defaults.

To select an element to edit without going into the HTML document, right click on the screen and click 'inspect'. On a tablet, use the finger or stylus to hilight the desired element.

With the /custom.css file, it is easy for a user to add a custom color pallette, remove or add background images, and even import new fonts. The custom.css file is the same as any other css file except that it will overwrite existing css rules contained within other documents.

In style.css, all stylesheets are called using @import. Because custom.css is at the bottom of the stylesheet, it will overwrite all other styles, allowing users to customize the color palette and clipart images without breaking the structual CSS with any potential editing errors.

A random color palette genorator was used to create the 'winter themed' default color palette and the background images were found on CleanPNG. These colors and images can easily be replaced in custom.css.

When a developer has chosen or created their own custom colors and images, they should move on to How To Add Content To This Template.
