How To Download This Template

When a user clicks "Download" on the navigation bar, they will be directed to a page with a large block of text inside a textarea. This is the user liscense. Users should read the liscense thoroughly before clicking the download links.

Below the liscence, there will be a few different download options. This page contains a brief guide for each type of download link.

Github

It is possible to download from github with or without an account. Simply click the bright green button within every repository to download the template as a zip file.

Some users may find it useful to 'fork' this template. Simply click the 'fork' icon and Github will do this automatically. This action requires an account.

Repl.it

As of right now, the only way to download using Repl.it is to have an account. Press the "fork" icon next to the green "run" button to copy the template. From there, the template can be downloaded as a zip file.
